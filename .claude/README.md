# Deskio Claude Instructions

Thư mục `.claude/` chứa documentation và context để hỗ trợ Claude AI làm việc hiệu quả hơn với Deskio codebase.

## Mục đích

Các file trong thư mục này giúp Claude hiểu rõ:
- Cấu trúc project và architecture
- Coding conventions và best practices
- Common workflows và patterns
- Database schemas và relationships
- API conventions
- Testing strategies
- Troubleshooting guides

## Cấu trúc

- `project-structure.md` - Tổng quan cấu trúc project
- `coding-conventions.md` - Quy ước code và best practices
- `database-schema.md` - Database schemas và relationships
- `api-patterns.md` - API design patterns
- `common-workflows.md` - Các workflows thường dùng
- `testing-guide.md` - Hướng dẫn viết tests
- `deployment-guide.md` - Quy trình deployment
- `troubleshooting.md` - Giải quyết các vấn đề thường gặp

## Sử dụng

Các file này được AI Agent tự động đọc để cải thiện context khi:
- Implement features mới
- Debug issues
- Review code
- Refactor code
- Viết documentation

## Cập nhật

Nên cập nhật các file này khi:
- Thêm patterns mới
- Thay đổi architecture
- Cập nhật conventions
- Thêm workflows mới
- Học được lessons learned từ bugs/issues
